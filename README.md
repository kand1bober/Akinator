# Akinator

Консольная реализация игры **Akinator** на C/C++, где программа пытается угадать персонажа или объект по ответам пользователя **yes / no**. Проект построен на базе бинарного дерева решений: внутренние узлы хранят вопросы/признаки, а листья — конкретных персонажей.

## Возможности

- запуск классической игры-угадайки;
- сохранение базы знаний в файл;
- построение дампа дерева в формате Graphviz;
- описание объекта по признакам;
- сравнение объектов;
- обучение на новых персонажах.

## Структура проекта

```
Akr/   — игровая логика
Tree/  — библиотека бинарного дерева
```

## Как это работает

Программа использует бинарное дерево:

- узлы — вопросы (признаки);
- листья — объекты.

Во время игры пользователь отвечает yes/no, и программа спускается по дереву. Если ответ неверный — добавляется новый объект и отличающий его признак.

## Режимы работы

- `p` — игра
- `s` — сохранить базу
- `d` — визуализация дерева
- `t` — описание объекта
- `c` — сравнение объектов
- `q` — выход

## Зависимости

- g++
- make
- SDL2
- readline
- Graphviz

## Сборка

```bash
make
```

Запуск:

```bash
make run
```

Очистка:

```bash
make clean
```

## Формат данных

База знаний хранится как дерево в текстовом формате со скобками.

Пример:

```
{
"животное"
    {
    "млекопитающее"
        {
        "кот"
            { "nuull" }
            { "nuull" }
        }
        {
        "рыба"
            { "nuull" }
            { "nuull" }
        }
    }
    { "nuull" }
}
```

## Визуализация

Генерируется `.dot` и `.svg` через Graphviz.

## Особенности

- собственная реализация дерева;
- использование указателей на родителей;
- цветной вывод в терминал;
- аудио через SDL2.
